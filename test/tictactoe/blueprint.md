# Tic-Tac-Toe Game

A simple single-file HTML tic-tac-toe game.

## Requirements

- Display a 3x3 grid for the game board
- Two players: X and O
- X goes first
- Players take turns clicking empty cells to make their move
- Detect when a player wins (three in a row, column, or diagonal)
- Detect when the game is a draw (all cells filled, no winner)
- Display the current player's turn
- Display the winner or draw status
- Include a "New Game" button to reset the board
- Use a clean, modern visual style with CSS
- All code in a single index.html file
